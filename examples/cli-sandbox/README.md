# CLI Sandbox

This folder is a fixed read-only sandbox for the NeuroCore CLI demo.

It contains a few small files that simulate a tiny engineering workspace:

- `project-overview.md`: project context and service dependencies
- `incidents/payments-latency.md`: incident notes
- `logs/deploy.log`: recent deploy history
- `configs/service-config.yaml`: selected runtime configuration

The CLI demo tools are intentionally limited to this folder.
