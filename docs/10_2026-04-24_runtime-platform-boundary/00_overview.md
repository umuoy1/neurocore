# Runtime 平台职责边界（2026-04-24）

本目录用于明确一件事：为了实现“超级强个人助理”，哪些能力必须在 `NeuroCore` 这个通用运行时平台里实现，哪些能力应留给基于 `NeuroCore` 构建的个人助理产品。

当前目录只回答平台侧问题，不展开产品侧实现细节。

## 使用方式

- 当需要判断“这个能力应该做在 NeuroCore 里，还是做在个人助理产品里”时，先读本目录。
- 当需要规划后续实现顺序时，以“是否具有通用复用价值”为首要判断标准。
- 当设计发生变化时，先更新本目录，再推进实现。
